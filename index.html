TaskPro API Test ve QA Raporlama

Bu depo, TaskPro uygulamasının API test otomasyonu ve Kalite Güvence süreçlerine ilişkin tüm belgeleri, test senaryolarını ve raporları içerir. Amacımız, API'nin güvenilirliğini ve performansını sağlamaktır.

Dosya Açıklamaları

  • ✓

    `Checklist.pdf`

    Manuel olarak yürütülen testlerin planlarını ve durumunu içerir.

  • ✓

    `Jira Bug Report.pdf`

    Proje sırasında bulunan ve hata takip sistemi (Jira) aracılığıyla raporlanan tüm hatalara ilişkin belgeleri içerir.

  • ✓

    `Performance Test Report.pdf`

    API'nin yük ve stres altındaki davranışını analiz etmek için yapılan testlerin sonuçlarını içerir.

  • ✓

    `Postman-Newman Report.pdf`

    Newman aracılığıyla Postman koleksiyonlarını çalıştırarak elde edilen otomasyon test raporlarını içerir.

  • ✓

    `Test Report.pdf`

    Tüm test türlerinin sonuçlarını bir araya getiren ve projenin genel test durumunu özetleyen kapsamlı bir rapordur.

  • ✓

    `TestRail-Cases Report.pdf`

    TestRail platformunda oluşturulan test senaryolarına ve çalıştırılan testlere ilişkin raporları içerir.

  • ✓

    `TestRail-RUN Report.pdf`

    TestRail üzerinde yürütülen test koşullarına (run) ilişkin sonuç raporlarını içerir.
